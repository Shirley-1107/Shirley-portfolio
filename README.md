## Tech Stack
- HTML5 + CSS3 (Flexbox, Grid, Glassmorphism, Media Queries)
- Vanilla JavaScript (form handling, scroll reveal)
- Google Fonts (Orbitron + Inter)
- GitHub Pages (free static hosting)
- Google Apps Script (serverless form backend → Google Sheets)

## Features
- Responsive & mobile-first design
- Cyber-themed UI with gradient background & glow effects
- Functional contact form (messages saved to Google Sheet)
- Hobbies section with personal photos & descriptions

Made with passion & purpose.
